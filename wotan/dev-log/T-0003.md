# T-0003: Verify SND partiprogram access, formats, and license

**Status**: BLOCKED | **Size**: M
**Phase**: —

## Blocker (2026-04-23)
Same as T-0002: background subagent can't reach external hosts (snd.se etc.). Deferred to main-agent context.

## Context
Programkompassen (T-0018) depends on being able to ingest Swedish party programs and election manifestos. SND hosts a collection ("Svenska partiprogram och valmanifest") but we must verify access, file formats, licensing, and practicality of ingestion before building the demo.

## Acceptance Criteria
- [ ] Access verified: at least three party programs downloaded to `data/samples/snd/`, covering at least two different parties and two different years.
- [ ] File formats documented in `docs/data/snd.md` (PDF? text? XML? metadata?).
- [ ] Licensing / attribution requirements documented.
- [ ] An ingestion sketch in `src/lib/ingest/snd.ts` (or worker) that parses the downloaded files into plain text with document metadata, with Vitest fixtures.
- [ ] If programmatic access is not practical, document a fallback (manual curation of a smaller set).

## Approach
1. Visit SND's collection page and document how documents are accessed.
2. Fetch a sample set via the documented channel.
3. Extract text, identify document boundaries, record metadata (party, year, title).
4. Note any restrictions.

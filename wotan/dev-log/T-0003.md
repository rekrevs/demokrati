# T-0003: Verify SND partiprogram access, formats, and license

**Status**: DONE | **Size**: M
**Phase**: —

## Context
Programkompassen (T-0018) depends on being able to ingest Swedish party programs and election manifestos. SND hosts a collection ("Svenska partiprogram och valmanifest") but we must verify access, file formats, licensing, and practicality of ingestion before building the demo.

## Acceptance Criteria
- [x] Access verified: 3 documents downloaded (S valmanifest 2022, M valmanifest 2022, MP valmanifest 2018) in both TXT and PDF.
- [x] File formats documented in `docs/data/snd.md`: `/sv/vivill/file/{party}/{type}/{year}/{txt|pdf}` URL pattern.
- [x] Licensing / attribution documented: SND itself has no formal CC licence on the collection; party texts are the parties' own IP, practical use = short quoted evidence + analysis under fair-use framing with attribution to both SND and the party.
- [ ] Ingestion code in `src/lib/ingest/snd.ts` — **deferred to T-0018**; docs include the design.
- [x] Fallback mapped: official party websites enumerated for all 8 riksdag parties.

## Approach
1. Investigated `snd.se/sv/vivill` + `/sv/vivill/party/{p}` + `/sv/vivill/party/{p}/{type}/{year}` pages.
2. Discovered direct download URL pattern via inspecting the document landing page.
3. Downloaded 3 samples in both formats to verify quality and size.
4. Checked for license statement on SND's collection/TOS pages.

## Log
- Found download URL pattern: `/sv/vivill/file/{party}/{type}/{year}/{txt|pdf}`.
- TXT files are clean UTF-8 with BOM, paragraph-structured, no OCR artifacts. Ideal for ingestion.
- PDF files range from ~130 KB to ~1.9 MB (the S valmanifest is design-heavy).
- No auth or rate-limiting hit during the three sequential downloads.
- SND's own terms-of-use don't explicitly licence the Vi vill collection — the collection is framed as research data from parties' own websites. Practical usage path: quote short evidence passages, attribute SND + party.

## Outcome
Primary ingestion path (direct SND TXT download) is viable and fast. Fallback (parties' own websites) is mapped in `docs/data/snd.md`. Programkompassen is not blocked.

Sample files saved:
- `socialdemokraterna-valmanifest-2022.txt` 32 KB (+ PDF 1.9 MB)
- `moderaterna-valmanifest-2022.txt` 78 KB (+ PDF 442 KB)
- `mp-valmanifest-2018.txt` 51 KB (+ PDF 128 KB)

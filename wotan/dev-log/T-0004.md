# T-0004: Add demokrati.janson.org A-record via HostUp UAPI

**Status**: DONE | **Size**: S

## What
Use HostUp UAPI to add DNS A-record for `demokrati.janson.org` pointing to the VPS IP `64.112.126.34`. Verify propagation with `dig`.

## Log
- Extracted SOA serial `2026012700` from `DNS/parse_zone` output.
- Posted `DNS/mass_edit_zone` with `add={"dname":"demokrati","ttl":3600,"record_type":"A","data":["64.112.126.34"]}`.
- Response `{ status: 1, data: { new_serial: "2026042400" } }` — accepted.
- `dig +short demokrati.janson.org @primary.ns.hostup.se A` → `64.112.126.34` (propagated to HostUp nameservers within seconds).

## Outcome
`demokrati.janson.org` now resolves to the VPS. Ready for Caddy TLS provisioning in T-0011.

Curl used (preserved for future reference — `hostup-api` script doesn't URL-encode complex values):
```bash
TOKEN=$(cat ~/.config/hostup/api-token)
curl -s -G -H "Authorization: cpanel fsfjm:${TOKEN}" \
  --data-urlencode "zone=janson.org" \
  --data-urlencode "serial=2026012700" \
  --data-urlencode 'add={"dname":"demokrati","ttl":3600,"record_type":"A","data":["64.112.126.34"]}' \
  "https://mu.hostup.se:2083/execute/DNS/mass_edit_zone"
```

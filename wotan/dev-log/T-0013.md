# T-0013: Build Riksdagsradarn

**Status**: BLOCKED | **Size**: L
**Phase**: —

## Context
Traceable summariser of a week of riksdag debates. Every AI claim clicks back to the exact speech segment. Switchable summary regimes: neutral / conflict / citizen.

## Acceptance Criteria
- [ ] Hybrid retrieval (BM25 + pgvector) over anföranden for a chosen date range and topic.
- [ ] Claims extracted with type (empirical / normative / critique / proposal) and linked to source chunk ids.
- [ ] Three summary regimes renderable from same claim set.
- [ ] UI at `/demo/riksdagsradarn` with filter/summary/source-card panels.
- [ ] At least 5 featured cases: skola, kriminalitet, energi, Nato/försvar, sjukvård.
- [ ] Confidence indicators (high/medium/low) displayed.
- [ ] Permalink + MethodsDrawer + localisation.
- [ ] E2E test loads a featured case under 2s; live run under 30s.

## Approach
Ingest nightly via worker job. Store chunks with embeddings. Retrieval pipeline first, then claim extraction, then regime rendering.

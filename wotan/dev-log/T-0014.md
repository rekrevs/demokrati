## Context
Visualise what a disagreement is really about. Axes are **inferred** from the corpus, not hardcoded. Transformationsmotorn's "what parties share / where they talk past each other / better next question" output is merged into a deliberative analysis tab.

## Acceptance Criteria
- [ ] Per-issue corpus (curated or user-supplied statements).
- [ ] Embed and cluster statements.
- [ ] Model infers 3–6 latent disagreement dimensions with left/right labels from the corpus.
- [ ] Axis switcher lets user pick which dimensions to render as X/Y of the scatter plot.
- [ ] Scatter plot with clusters coloured and tooltips showing statement text.
- [ ] Deliberative analysis tab: shared premises, empirical disagreements, value conflicts, contested terms, better next question.
- [ ] At least 6 featured corpora.
- [ ] Permalink + localisation.
